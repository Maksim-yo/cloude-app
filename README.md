# Cloude file storage

Service for storing files

## Functionality
- Sign Up
- Log out
- Log in
- Uploading
- Creating
- Deleting
- Renaming

## Technology stack
- Bootstrap + JQuery
- Django
- PostgreSQL
- Minio
- Docker for deployment
## Deploy

```shell
  docker-compose --env-file .env -f docker-compose.yml up
```
